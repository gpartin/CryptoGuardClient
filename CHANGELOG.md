# Changelog

All notable changes to `CryptoGuardClient` will be documented in this file.

## [0.3.0] — 2026-06-26

### Added
- Initial public release of CryptoGuardClient SDK
- Python client with typed exception hierarchy
- 5 MCP tools: validate_trade, scan_token, rug_check, search, health
- MCP server with stdio + HTTP transports
- Smithery, Glama, and awesome-mcp-servers configuration
- PyPI publishing via GitHub Actions (trusted publisher)
- Free tier: 5 calls/day per IP, no signup required
- x402 USDC micropayments for unlimited access ($0.05/call)
- Contract address resolution across 7 chains
- Batch validation for up to 20 trades
- DexScreener-powered rug pull detection
- Portfolio scanning for up to 50 tokens
